A WordPress security and hardening plugin from SMHeart Inc.  

== Description ==

The Administrative User is presented with several independently selectable options for making their default installation increasingly secure.
* Remove wp_generator meta information
* Remove wlwmanifest_link meta information
* Remove rsd_link meta information
* Enable Comment HTML filter


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder 'smheart-security' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the SMHeart Security settings page to configure plugin.
